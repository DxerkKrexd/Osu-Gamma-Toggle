# Osu-Gamma-Toggle
A tool to toggle gamma on windows so you don't have to tab in and out

IF YOU HAVE ANY ISSUES PLZ ADD MY DISCORD SO I CAN (prolly cant) HELP YOU

You NEED python 3.10 or higher.


To Toggle Its F8, to increase/decrease the value its +,-


You can CHANGE the keybind using Edit in IDLE, Found by right clicking the .py file

On the most RECENT version there IS no ui. It runs in the tray. So just open the tray and right click it to close it. If you WANT the ui just run the ,py file (either one i think is ok)

To open Just double click the .py file, OR you can open cmd and drag it into there and exec it



To CHANGE the color of the ASCII art (idk why you'd want to do that but wtv) right ABOVE the ascii it should say "{Fore.CYAN}{Style.DIM}" you can change the CYAN to "Fore.RED, Fore.GREEN, Fore.BLUE, Fore.CYAN, Fore.MAGENTA, Fore.YELLOW, and Fore.WHITE".


The styles that you can put include Style.BRIGHT, and Style.DIM. If you dont want either you can just delete it from the line
